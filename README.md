&emsp;&emsp;本项目用于记录比赛，包括部分论文复现。

## Detail
- [x] 使用 **position** 特征
- [x] 使用 **postag** 特征
- [x] 使用 **hypernym** 特征
- [x] 固定窗口使用 **entity** 特征
- [x] 字向量使用 **sgns (negative sampling)**
- [x] 词向量使用 **topic word embedding** 和 **glove**
- [x] 字词向量融合使用 **add** 形式
- [x] 使用 **3-grams concat (RNN)** 输入 
- [x] 对 **NA** 做特殊处理, 包括使用 **grid search** 阈值判断
- [x] **不采用** softmax cross-entropy 作为损失函数
